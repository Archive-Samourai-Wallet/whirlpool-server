# whirlpool-server
Whirlpool server

## Usage
- edit server configuration:
```
application.properties
```

- run from IDE:
```
com.samourai.whirlpool.server.Application
```

- run from commandline:
```
mvn clean install -Dmaven.test.skip=true
java -jar target/whirlpool-server-0.0.1-SNAPSHOT.jar
```
